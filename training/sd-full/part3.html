Mini-Challenge: FizzBuzz (the classic interview question)

Print numbers 1 to 30. For multiples of 3 print "Fizz", multiples of 5 print "Buzz", multiples of both print "FizzBuzz". Hint: the remainder operator — number % 3 === 0 means divisible by 3. Solve it before peeking online; it uses only this part's tools.

3.8 Checkpoint Project — Console Student Manager

Build part3-js/student-manager.js that does all of the following when run with node. Use only this part's concepts:

  1. Stores an array of at least 6 student objects (name, score, feesPaid true/false).
  2. A function addStudent(name, score, feesPaid) that pushes a new student.
  3. Prints the class average, the highest scorer's name, and the pass rate (score >= 50).
  4. Prints the names of students who passed but have NOT paid fees (filter with &&).
  5. A function gradeOf(score) returning "A" (70+), "B" (60+), "C" (50+) or "F", and a printed report line per student: "Ada — 82 — A — PAID".
CHECKPOINT

If you built this without copying, you are now — genuinely — a programmer. Everything from here is applying this thinking to bigger surfaces.
