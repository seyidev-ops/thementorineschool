7.1 Why a Second Language Doubles Your Value

Python dominates automation, data analysis, scripting, and AI — the areas JavaScript touches least. More importantly, learning a second language proves to yourself that you learned programming, not just JavaScript. The concepts are identical: only the spelling changes. This part moves fast on purpose — you already know how to think.

ConceptJavaScriptPython
Variableconst name = "Ada";name = "Ada"
Printconsole.log(x)print(x)
Functionconst f = (a, b) => a + b;def f(a, b): return a + b
Conditionif (x > 5) { … } else { … }if x > 5: … else: …

7.6 Checkpoint Project — Expense Analyser CLI

Build expenses.py: a command-line tool that reads expenses.csv (date, category, amount) and prints: total spend, spend per category sorted highest first, the single biggest expense, and the average per day. Bonus: accept a category name as a command-line argument (python expenses.py food) and filter the report — hint: import sys then sys.argv.
