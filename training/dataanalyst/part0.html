0.1 What a Data Analyst Actually Does

Every business drowns in records — sales, customers, deliveries, ad clicks — and starves for answers: Which product makes us money? Why did March collapse? Which customers are about to leave? A data analyst turns the records into the answers. The craft has four moves, repeated forever: ask a sharp question, prepare the data honestly, analyse it with spreadsheets, SQL and Python, and communicate the finding so a decision gets made.

This guide takes you from absolute zero to job-ready in a recommended 8–12 weeks, using an entirely free toolkit, and ends with the NaijaSales capstone: a full analysis of a messy retail dataset, delivered as an insight memo and a live dashboard — the exact artifacts hiring managers ask to see.

WHY THIS MATTERS

Analysts are hired in every industry — banks, telcos, retail, NGOs, startups — because every industry has data and decisions. You are not learning a tool; you are learning to be the person in the room who knows.

0.2 The Golden Rules of Analysis

Question first, tool second. An analysis without a sharp question is a tour, not a journey. Never trust raw data. Real data arrives dirty — duplicated, mistyped, half-missing; Part 6 makes you its janitor and judge. Show your working. Every number you publish must be reproducible: same data, same steps, same result. Speak human. A finding that isn't understood is a finding that didn't happen — charts and sentences beat tables and jargon.

0.4 Set Up Your Free Toolkit (Practical)

Everything in this course is free. (1) Google Sheets — any Google account; our spreadsheet home. (2) Python from python.org — during Windows install, tick “Add Python to PATH”. (3) VS Code from code.visualstudio.com for writing scripts. (4) Looker Studio at lookerstudio.google.com — free dashboards, same Google account. Verify Python, then install the two libraries of the trade:

python --version        # or python3 --version on Mac/Linux
pip install pandas matplotlib

Create your journey folder and download the course dataset — a year of sales from a fictional Lagos retailer, deliberately seasoned with real-world dirt:

mkdir analyst-journey
cd analyst-journey
mkdir part2-sheets part3-sql part5-pandas part6-cleaning part7-viz part9-capstone

The dataset (naijasales.csv) is generated by a script you will write in Part 5 — by then you'll understand every column in it: date, order id, product, category, region, unit price, quantity, customer id.

0.5 How Every Lesson Is Structured

Concept in plain English → Copy-and-Paste Practical with a FILE label → Try This modification → per-part Checkpoint. One promise: when a formula or query returns something surprising, you will not shrug — you will pick three rows and verify by hand. Hand-checking small samples is the analyst's lie detector, and it never retires.
